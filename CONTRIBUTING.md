# コントリビューション・動作報告について

このプロジェクトは個人開発として始まったものですが、動作報告・バグ報告・
プルリクエストを歓迎します。

## 動作報告（特に歓迎）

検証済みディストリビューション以外（Fedora, Arch, openSUSE等）で動かして
みた結果は、Issueでぜひ報告してください。以下の情報があると助かります。

- ディストリビューション名・バージョン
- デスクトップ環境（XFCE, GNOME, KDE等）
- `install.sh` 実行時のログ（`~/qji_install_debug.log`）
- 発生したエラーメッセージ

## バグ報告

再現手順と、可能であれば以下のログを添付してください。

```bash
cat ~/qji_install_debug.log       # インストールログ
cat ~/qji/library_build_debug.log # ライブラリー構築ログ（あれば）
cat /tmp/qji_*.log                # 各機能の実行時ログ
```

## プルリクエスト

- 大きな変更の前にIssueで方向性の相談をお願いします
- `install.sh` や `scripts/*.sh` の変更は、複数ディストリビューションへの
  影響を考慮していただけると助かります（ターミナルエミュレーターの違い、
  デスクトップパスの違い等）

## 既知の制約

- BlueALSA（Bluetooth）経由の再生は、ディストリビューション・カーネル
  バージョンによって動作しないことがあります（詳細はコード内コメント参照）
- Vosk音声認識モデルは別途ダウンロードが必要です
